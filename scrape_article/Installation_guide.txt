Create environment inside scrape_article folder:
For Win: python -m venv env
For MacOS: virtualenv venv

Activate environment:
For Win: .\env\Scripts\activate
For MacOS: source venv/bin/activate

Installing all required packages: pip install -r requirements.txt

Enter article_scrapper folder: cd article_scrapper

Run the script: scrapy crawl article -o output.json -t json
